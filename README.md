# zenn_exported_articles

Zennの `@funatsufumiya` アカウント閉鎖に伴うバックアップです。

[articles](./articles)以下にあるMarkdownより内容閲覧が可能ですが、Markdownファイル名はタイトルではなくハッシュ値のため、目次がないのが現状です。（目次の生成は検討中です。）

## 補足（アカウント閉鎖の経緯）

一種のNo-AI活動の一環としてアカウントを閉鎖したものです。

私自身過去にCNN等のディープラーニングによる画像認識等を行っていたこともあり、AI利用自体は完全否定派ではありませんが、AI記事があまりに多く、業界全体としてもAI推進の動きが見られるため、その反発として閉鎖したものです。

ただ、AI自体の社会的意義は理解しているため、こちらのリポジトリに残っているデータのAI学習は禁止はしません。

[Cara](https://cara.app/) のような技術コミュニティがもしできれば、内容を吟味しながらそちらに記事を移すことは検討したいと思っています。
